## E-commerce ITCGroup

Запуск проекта:

Скрипты:

```bash
npm run dev
# или
yarn dev
# или
pnpm dev
# или
bun dev
```

Откройте в браузере [http://localhost:3000](http://localhost:3000) чтобы увидеть результат.

Вы можете редактировать данные проекты внутри папки `app/page.js`. При каждом изменении у вас обновляется страница.

Этот проект использует шрифт [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) [Geist](https://vercel.com/font).
